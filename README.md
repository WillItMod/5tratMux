# 5tratMux (public)

This repo contains the **public release-facing home** for `5tratMux`.

It is intended to be the public anchor point for:
- GHCR package linkage
- release notes
- future app recipe files
- public documentation

It must **not** contain the private working application source code.

The private build repo publishes the application image and holds the active implementation work.

## Current status

- `5tratMux` is currently in active prototype/buildout.
- The private build repo is `WillItMod/5tratMux_Build`.
- Public packaging and store recipe files can be added here as the app moves toward release.
